## 📝 购买步骤（以阿里云为例）

### 1. 注册账号
访问：https://www.aliyun.com/
点击 "免费注册"

### 2. 实名认证
- 个人认证：身份证 + 人脸识别
- 企业认证：营业执照

### 3. 选择产品
进入 "轻量应用服务器" 页面

### 4. 选择配置
- **地域**：选择离您最近的（如华东1-杭州）
- **镜像**：Ubuntu 22.04
- **套餐**：2核2G 3M带宽
- **购买时长**：建议 3 个月（可测试）

### 5. 支付
支持支付宝、微信支付、信用卡

### 6. 配置服务器
- 设置 root 密码
- 开放防火墙端口（80, 443, 22）

### 7. 获取 IP
在控制台查看公网 IP 地址

---

## 🔐 安全提示

1. **立即修改默认密码**
